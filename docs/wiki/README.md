## Resumen del Sistema

Tu Visión es un sistema de gestión para ópticas que permite:

1. **Gestión de Usuarios**: Administradores y vendedores con diferentes niveles de acceso
2. **Órdenes de Trabajo**: Registro de ventas de armazones, lentes, y lentes completos
3. **Recetas Médicas**: Almacenamiento de prescripciones ópticas
4. **Auditoría**: Seguimiento de quién creó y modificó cada registro
